# flashcard

Mobile-friendly flashcard MVP: Next.js (App Router), TypeScript, Tailwind CSS, localStorage. Run with [pnpm](https://pnpm.io).

## Scripts

```bash
pnpm install
pnpm dev
pnpm run build
pnpm run test        # Vitest (watch)
pnpm run test:run    # Vitest once
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

- Next.js 16 · React 19 · Vitest · Testing Library
